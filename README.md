# 👕 CoFoVo | Boutique Handmade Wear

Welcome to the official repository of **CoFoVo**. 

## ✨ About the Brand
CoFoVo is a family-run boutique clothing brand. Our mission is to combine authentic craftsmanship with modern style. 

- **Handmade Quality:** Every single piece in our collection is carefully sewn and handcrafted by my mother.
- **Sustainable Fashion:** We support slow fashion by creating durable, high-quality items instead of mass-production.
- **Family Business:** Operated by a young entrepreneur and a master seamstress.
